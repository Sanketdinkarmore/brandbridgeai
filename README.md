# 🌉 BrandBridge AI

> Stop competing for budget. Start collaborating for growth.

🌍 **Live Demo:** [https://brandbridgeai.vercel.app/](https://brandbridgeai.vercel.app/)

BrandBridge AI is a premium SaaS platform that enables intelligent brand collaborations, cross-promotion, and AI-generated campaign workflows. By leveraging advanced matching algorithms (powered by Gemini), we pair highly compatible brands together, automatically draft conversion-ready outreach proposals, generate cross-platform campaign assets, and seamlessly integrate vetted freelance creatives to bring shared visions to life.

## 🚀 Core Features

- **AI-Powered Brand Compatibility:** Our engine evaluates audience overlap, industry alignment, and shared goals to generate precise 0-100% Match Scores between businesses.
- **Automated Campaign Generation:** Skip the blank page. Our AI actively generates cross-platform ad variants (Instagram, LinkedIn, Facebook, Google Ads) based on the combined identity of the matched brands.
- **Freelancer Marketplace:** Verified creatives (Designers, Editors, Copywriters) can be sourced and hired directly into active collaboration projects.
- **Cost-Splitting Visibility:** See exactly how marketing expenses break down in real-time, enabling brands to double their reach while securely splitting the financial lift.
- **Role-Based Workflows:** Distinct dashboard environments tuned perfectly for Brands, Product Owners, and Freelance Creatives.

## 🛠 Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **UI & Styling:** TailwindCSS & custom premium glassmorphism scaling (`.bb-glass`, `.bb-glass-intense`)
- **Icons:** Lucide React
- **AI Integration:** Google Gemini API
- **Database:** MongoDB & Mongoose

## 💻 Getting Started

First, install dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to experience the platform.
